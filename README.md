# Ecom-App (productcatalog)

Java Spring Boot backend and React (Vite) frontend for a product catalog.

Quick start:
- Backend: cd /Users/dhananjaysingh/Downloads/productcatalog && ./mvnw spring-boot:run
- Frontend: cd /Users/dhananjaysingh/Downloads/productcatalog/ecom-catalog-react && npm install && npm run dev
